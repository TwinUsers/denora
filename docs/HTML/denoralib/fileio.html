int FileExists(char *fileName)
Checks if the file list exists, and returns 1 on finding the file, or 0 on failure, and sets DenoraLastErrorCode

int FileRename(char *fileName, char *newfilename)
Renames the file to the new filename, if error sets DenoraLastErrorCode

FILE *FileOpen(char *file, const char *mode)
Opens a file pointer to the file, no need for ifdef the function works fon *nix and win32 systems, if error sets DenoraLastErrorCode
